Declaratie Module Cookie Ce sunt, cum si ce cookies folosim pe site-ul nostru WWW ( World Wide Web ) este construit pe o premisă foarte simplă, dar puternică. Toate materialele de pe Web sunt formatate într-un format general, uniform, numit HTML ( Hypertext Markup Language ), iar toate solicitările de informații și răspunsurile sunt conforme unui protocol standard similar. Când cineva accesează un server de pe Web, cum ar fi siteul nostru fotovideonuntabucuresti.ro, browserul Web al utilizatorului va trimite o solicitare de informații la computerul serverului unde siteul este gazduit. Serverul Web va răspunde cererii prin transmiterea informațiilor dorite către computerul utilizatorului. Acolo, browserul utilizatorului va afișa informațiile primite pe ecranul utilizatorului. Pe tot acest parcurs, apar Cookies ( prajiturele ) Ce este un cookie? Un cookie este o informație sub forma unui fișier text foarte mic care este plasat pe un hard disk al unui utilizator de internet. Acesta este generat de un server de pagini web, care este în principiu computerul care operează un site web. Informațiile conținute de cookie sunt setate de server și pot fi utilizate de acel server ori de câte ori utilizatorul vizitează site-ul. Un cookie poate fi considerat ca un card de identitate al utilizatorului de internet, care spune unui site web atunci când utilizatorul sa întors. Sunt cookie-urile periculoase? Nu. Cookie-urile sunt mici bucăți de text. Nu sunt programe de calculator și nu pot fi executate ca si cod. De asemenea, ele nu pot fi utilizate pentru a difuza viruși, iar versiunile moderne ale browserelor Chrome, Firefox, Internet Explorer, Safari, Netscape, Opera, le permit utilizatorilor să-și stabilească propriile limitări în ceea ce privește numărul de cookie-uri salvate pe hard disk-urile lor. Pot cookie-urile amenința confidențialitatea utilizatorilor? Cookie-urile sunt stocate pe unitatea hard disk a computerului dar nu pot accesa unitatea de disc, prin urmare un modul cookie nu poate citi alte informații salvate pe unitatea de hard disk cum ar fi adresa de e-mail a unui utilizator etc. Ele conțin și transferă la server doar informațiile pe care le-au dezvăluit utilizatorii înșiși unui anumit site web. Un server nu poate seta un cookie pentru un domeniu în care nu este membru. Cu toate acestea, utilizatorii găsesc adesea în fișierele lor cookie-uri de pe site-uri web pe care nu le-au vizitat niciodată. Aceste cookie-uri sunt de obicei stabilite de către companiile care vând publicitate pe internet în numele altor site-uri web. Prin urmare, este posibil ca informațiile utilizatorilor să fie transmise către site-uri terțe, fără cunoștințele sau acordul utilizatorilor, cum ar fi informații despre obiceiurile de navigare. Acesta este motivul cel mai frecvent pentru care persoanele resping sau se tem de cookie-uri. Istoricul cookie-urilor Cookie-urile pentru internet au fost inițial dezvoltate în 1995 de către Netscape Communications Corporation. Cuvântul “cookie” vine de la “cookie-ul magic”, un termen în limbile de programare, pentru o informație partajată între componentele software care cooperează. Alegerea cuvântului cookie pare să provină din tradiția americană de a oferi și de a împărți prajiturele. Care este scopul cookie-urilor? Modulele cookie fac interacțiunea dintre utilizatori și site-uri web mai rapidă și mai ușoară. Fără cookie-uri, ar fi foarte dificil pentru un site web să permită unui vizitator să umple un cos de cumpărături sau să-și amintească preferințele utilizatorului sau detaliile de înregistrare pentru o vizită ulterioară. Site-urile Web folosesc cookie-uri în principal pentru că economisesc timp și fac experiența de navigare mai eficientă și mai plăcută. Site-urile Web utilizează adesea cookie-uri în scopul colectării informațiilor demografice despre utilizatorii lor. Cookie-urile permit site-urilor web să monitorizeze obiceiurile de navigare pe web ale utilizatorilor și să le profileze în scopuri de marketing (de exemplu, pentru a afla care produse sau servicii care le interesează și pentru a le trimite reclame vizate. Cookie-urile vin în diferite “arome”: Sesiune sau cookie-uri tranzitorii Cookie-urile care sunt stocate în memoria calculatorului numai în timpul sesiunii de navigare a unui utilizator și sunt șterse automat de pe computerul utilizatorului când browser-ul este închis. Aceste cookie-uri stochează de obicei un ID de sesiune care nu este identificabil personal de către utilizatori, permițând utilizatorului să se mute de la o pagină la alta, fără a fi necesar să vă conectați în mod repetat. Acestea sunt utilizate pe scară largă de site-urile web comerciale (de exemplu, pentru a urmări elementele pe care un consumator le-a adăugat unui coș de cumpărături). Cookie-urile pentru sesiuni nu sunt scrise niciodată pe hard disk și nu colectează nicio informație de la computerul utilizatorului. Cookie-urile pentru sesiuni expiră la sfârșitul sesiunii de browser a utilizatorului și pot să nu mai fie accesibile după ce sesiunea a fost inactivă pentru o anumită perioadă de timp, de obicei 20 de minute. Cookie-uri permanente, persistente sau stocate Cookie-urile care sunt stocate pe computerul utilizatorului și nu sunt șterse când browserul este închis. Cookie-urile permanente pot păstra preferințele utilizatorilor pentru un anumit site web, permițând aceste preferințe să fie utilizate în viitoarele sesiuni de navigare. Cookie-urile permanente pot fi folosite pentru a identifica utilizatorii individuali, astfel încât aceștia pot fi utilizați de site-uri web pentru a analiza comportamentul utilizatorilor în navigarea pe site-ul web. Aceste cookie-uri pot fi, de asemenea, utilizate pentru a furniza informații despre numărul de vizitatori, timpul mediu petrecut pe o anumită pagină și, în general, despre performanța site-ului Web. Ele sunt, de obicei, configurate pentru a urmări utilizatorii pentru o perioadă prelungită de timp, în unele cazuri mulți ani în viitor.
